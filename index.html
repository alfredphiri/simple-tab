Vanilla JS SPA Demo

This is a minimal Single Page Application (SPA) built with pure HTML, CSS, and JavaScript — no React, no Vue, no frameworks at all.

Click the tabs above to navigate between views without reloading the page. That is the core idea behind a SPA!

What is a SPA?

A Single Page Application loads one HTML page and dynamically updates content as the user interacts — without full page reloads.

This demo achieves that by toggling the visibility of div sections using JavaScript display style changes — simple and effective.

How it works

1. Hide all sections

When a tab is clicked, a hideAll() function sets every content section's display to none.

2. Show the active section

The section matching the clicked tab is then set to display: block, making it visible.

3. Track the active tab

An .active CSS class is toggled on the clicked tab to highlight the current view.

4. Zero dependencies

This entire routing mechanism uses native browser APIs — no libraries needed.

Try it yourself

Open index.html in any browser — no build step, no install, no server required.

Fork the source and extend it with window.history.pushState() to add real URL-based routing.
